To build the examples, run `npm i && node examples/example`

### Labels

```
||This is a label!||
||This is a label!|warning||
||This is a label!|success||
```

### Spoiler

```
> and captcha protected spoilers:

\{{Simple spoiler|info}}

!\{{Write me at <john.doe@example.com>|info}}
```

### Progress bars

```
=====0%=====
=====10%|error=====
=====40%|warning=====
=====80%|success=====
```

### Glyphicons

```
::|search::
```

### Badges

```
[[42]]
[[42|danger]]
```

### Jumbotrons

```
////

```
